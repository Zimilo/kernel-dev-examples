提供了针对最新内核的一系列kernel开发相关的例子。供大家学习内核的时候参考，同时也包含了我写的一系列文章 learning linux kernel的源代码。

因内核代码更新较快，这里的例子会尽量保持与最新的内核(指linus最新的stable的git内核版本)代码同步。

当前更新到的版本为3.6.2

如果发现某些例子有问题，可以给我mail: zimilo#code-trick.com,也欢迎交流内核开发相关内容。
